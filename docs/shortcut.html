描述 操作方式 备注

运行命令 win + r $1

关闭窗口 ctrl + f4 $1

创建虚拟桌面 ctrl+win+d
切换虚拟桌面 ctrl+win+方向
关闭虚拟桌面 ctrl+win+f4

锁定桌面 win+l 离开工位
