# puppeteer

文档地址：https://zhaoqize.github.io/puppeteer-api-zh_CN/

使 puppeteer 显示插件，控制台默认打开

提升开发机的 开发，调试效率

# 1.个人资料路径

浏览器地址输入：chrome://version 获得 chrome 浏览器的 “个人资料路径”

# 2.插件路径

插件路径：个人资料路径/Extensions/插件id/插件版本 (nodejs 里面的路径 需要把‘\’ 换成 ‘/’)

通过浏览器路径 chrome://extensions 找到需要开启插件的id

获得完整插件路径

# 3.puppeteer 配置

处理路径


 



 
 































const path = require('path')
const devtoolsPath = path.resolve(`${插件路径}`)

const CONFIG = {
  headless: false,
  args: [`--disable-extensions-except=${devtoolsPath}`],//启用插件
  devtools: true // 默认打开控制台，即便是打开新的标签页
};
const PAGE_INDEX_URL = ' http:localhost:8080'

(async () => {
  // 初始化实例
  const puppeteer = require("puppeteer");
  const browser = await puppeteer.launch(CONFIG);
  const page = await browser.newPage();
  await page.goto(PAGE_INDEX_URL)

  // 选择密码登录
  const submitBtn = await page.$('#submit')
  const userInputSelector = '#account'
  const userPswdSelector = '#password'
  
  // 确保 指定元素 dom 渲染完毕
  await page.waitForSelector(userInputSelector)
  await page.waitForSelector(userPswdSelector)
  
  // 输入账号密码，点击登录
  await page.type(userInputSelector,'1100000000',{
    delay: 10
  })
  await page.type(userPswdSelector,'123456',{
    delay: 10
  })

  await submitBtn.click()
})();
